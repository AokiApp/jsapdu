## How to Contribute

1. **Fork the repository**  
   Click the "Fork" button on GitHub and clone your fork locally.

2. **Create a branch**  
   Use a descriptive branch name, e.g. `fix/card-session-bug` or `feature/nfc-support`.

3. **Make your changes**  
   - Follow the existing code style (TypeScript, ESLint, Prettier).
   - Add or update tests as needed.
   - Document new features or changes.

4. **Test your changes**  
   Run the test suite:
   ```bash
   npm install
   npm run build
   npm test
   ```

5. **Commit and push**  
   Write clear, concise commit messages.

6. **Open a Pull Request**  
   - Go to your fork on GitHub and open a PR against the `main` branch.
   - Fill out the PR template and describe your changes.

## Code of Conduct

Please be respectful and constructive according to general open source community standards.

## Reporting Issues

- Use [GitHub Issues](https://github.com/AokiApp/jsapdu/issues) for bugs, feature requests, and questions.
- Search existing issues before opening a new one.

## Style Guide

- TypeScript, strict types
- Use async/await for asynchronous code
- Prefer functional, modular design
- Write clear JSDoc comments
